Natural Earth Downloader

Select one or several Natural Earth vector datasets, choose an output format and download them directly into QGIS.

Workflow

  1. Search or filter the catalogue.
  2. Check the datasets you need.
  3. Choose GeoPackage, GeoJSON, Shapefile or temporary layers.
  4. Choose an output location and start the batch.
Recommended: use GeoPackage for multi-layer downloads. It preserves long field names and keeps the batch in one file.

Temporary layers

Temporary layers live in memory and can be lost when the QGIS project closes. Save them from QGIS when you need permanent copies.

Cache

Downloaded source archives are cached in the QGIS profile. Repeating a download reuses the cached archive. The Clear cache button removes source archives but not your exported outputs.
